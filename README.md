# DisparityVisualizer
Simple 3D disparity visualizer

## Build Instructions

Create the `build` directory and enter it:

```
mkdir build
cd build
```

from build directory call `cmake` and `make`

```
cmake ..
make
```

it should be enough to create the executable.
